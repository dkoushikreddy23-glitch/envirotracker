# 🌍 Enviro Tracker

Enviro Tracker is a web application that provides real-time environmental information including:

- 📰 Latest News
- 🌦 Weather Updates
- 🌫 Air Quality Index (AQI)

## 🚀 Features
- Real-time data updates
- Clean and simple UI
- Easy to use interface

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 📌 Purpose
This project helps users stay informed about environmental conditions and make better decisions.
